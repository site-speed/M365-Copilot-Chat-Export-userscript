# M365 Copilot Chat Conversation Exporter — Userscript

Version: **v1.0.20**

Export the current Microsoft 365 Copilot Chat conversation to readable Markdown and raw JSON Markdown files.

## Output files

1. readable Markdown: `.md`;
2. raw JSON Markdown companion: `.json.md`.

## Release notes

Current public release notes are maintained at:

```text
assets/release-notes.md
```

The filename is stable so GitHub releases can use the same `gh release create ... --notes-file assets/release-notes.md` pattern for every version.

## Privacy

Exports are generated locally from your authenticated browser session and may contain sensitive organisation data.
