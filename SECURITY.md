# Security Policy

This project exports conversations from an authenticated Microsoft 365 Copilot Chat browser session.

## Supported versions

Only the latest released userscript snapshot is supported.

## Reporting issues

Please report security or privacy concerns privately where possible before opening a public issue.

## Privacy posture

- The public release repositories should contain only distribution artefacts and minimal support metadata.
- Private development transcripts, raw fixtures, archived docs, project logs, and governance history must not be published.
- Raw `.json.md` exports are the authoritative local backup and may contain sensitive content; treat them as private data.
