# M365 Copilot Chat Conversation Exporter Userscript v1.0.30

This userscript release keeps the exporter aligned with the shared project version.

## What it does

The userscript exports the current Microsoft 365 Copilot Chat conversation as readable Markdown with a raw JSON Markdown companion backup.

## Install for the userscript

Install from the public userscript source once the public repository or userscript listing has been refreshed for this version.

## Source and support

Source and issue reporting are available from the public userscript repository.
